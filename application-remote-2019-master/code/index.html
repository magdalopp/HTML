What is HTML?

    HTML stands for Hypertext Markup Language and it’s the standard markup language for creating web pages and web applications. HTML describes the structure of the web page, with text, links, pictures, and even video, so the web browser can display them correctly. The HTML elements are the building blocks of HTML pages. It’s easy to use and provides you with structural semantics for texts such as headings, paragraphs, lists, links, quotes and other items.

    You can also use HTML if you want to add metadata to your webpage. The metadata is not always shown, it’s the information about the webpage and it can for example include the name of the person that has created the page.
